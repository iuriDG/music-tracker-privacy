# Extension Icons

## Creating Icons

You need to create three PNG icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Option 1: Use the HTML Generator
1. Open `create-icons.html` in your browser
2. Right-click each canvas image
3. Select "Save image as..."
4. Save with the corresponding filename (icon16.png, icon48.png, icon128.png) in this icons folder

## Option 2: Create Custom Icons
Use any image editor (Photoshop, GIMP, Figma, etc.) to create custom icons with the sizes above.

Design recommendations:
- Use a music-related icon (music note, headphones, etc.)
- Use a gradient background (purple/blue works well)
- Keep it simple and recognizable at small sizes
- Use white or light colors for the icon symbol

## Option 3: Use Online Tools
You can use online icon generators or converters:
- https://favicon.io/
- https://www.favicon-generator.org/
- Canva icon maker

## Temporary Workaround
The extension will work without icons, but Chrome/Firefox will show a default placeholder icon instead.
